# 🌊 Titan Visual Map: The Liquid Habitat

This map defines the spatial and visual "Habitat" for each media type within the Titan Ocean.

## 1. The Global Shell: The Volumetric Ocean
*   **Background**: Deep-blue gradient with real-time AGSL Caustics.
*   **Lighting**: Single global "Sun" source (top-right) using ray-tracing math for card shadows.
*   **Atmosphere**: Liquid viscosity (subtle blur on distant elements).

## 2. Manga Habitat: The Vertical Canyon
*   **Spatiality**: A deep, narrow trench where pages flow vertically.
*   **Interaction**: Frictionless "Infinite Scroll" with water-ripple effects on page boundaries.
*   **Visual**: Pages cast soft "Path-Traced" shadows against the canyon walls.

## 3. Anime Habitat: The Theater Bubble
*   **Spatiality**: A floating, transparent glass bubble in the mid-ocean.
*   **Interaction**: Cinematic focus mode with "Light Leak" refraction during high-action scenes.
*   **Visual**: Volumetric "Glow" emanating from the video player into the surrounding ocean.

## 4. Music Habitat: The Pulse Core
*   **Spatiality**: A centered, pulsing energetic core.
*   **Interaction**: Physics-based "Shockwaves" that distort the ocean surface on every beat.
*   **Visual**: Liquid particles that dance based on frequency analysis (FFT).

## 5. Webnovel Habitat: The Reading Stream
*   **Spatiality**: Clean, flat surface "floating" on the waves.
*   **Interaction**: Paper-like "Melt" transition when turning pages.
*   **Visual**: High-contrast, eye-comfort typography with subtle "Ink-Leak" animations.
